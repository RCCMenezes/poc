# Geomarketing Platform

Plataforma Dash para análise e comparação das redes Alcampo, Carrefour, Dia, Lidl e Mercadona em Espanha.

Cada cadeia disponibiliza três vistas:

- mapa de pontos;
- heatmap de densidade;
- clusters interativos que se expandem com o zoom.

A análise de interseções suporta todas as combinações únicas entre as cinco cadeias, com raio configurável.
