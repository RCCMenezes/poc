# Geomarketing Platform

Plataforma Dash para análise e comparação das redes Alcampo, Carrefour, Dia, Lidl e Mercadona em Espanha.

Cada cadeia disponibiliza três vistas:

- mapa de pontos;
- heatmap de densidade;
- clusters interativos que se expandem com o zoom.

O menu de análise espacial inclui também uma vista global de clusters, com todas as cadeias diferenciadas por cor.

O menu Estatísticas apresenta comparativos por cadeia, município e distrito.

O menu News reúne metadados do RSS público do Alimarket e liga cada notícia à fonte original.
As manchetes são atualizadas automaticamente a cada 15 minutos e também podem ser atualizadas manualmente.

A análise de interseções suporta todas as combinações únicas entre as cinco cadeias, com raio configurável.
